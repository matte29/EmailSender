Email Information Help

From: This is where you need to enter the email address you are sending the emails from.

Subject: This is where you need to enter what you want to be in the subject line of the emails.

Password: This is where you need to enter the password for the email address given above. For some email clients like Gmail / GSuite, you may need to disable secure login in the settings of the email client. To turn on Less secure app access from the settings(Manage your Google Account > Security > Less secure app access)

HTML File: Here you will click the button and select the HTML file from your computer. This will be used as the body of the email. Currently there is no support for selecting a file from Google Drive, OneDrive or any other cloud storage service.

CSV File

Field Name: This is the name of the Column of the Email from the SQL Database.

CSV File: Use standered Comma seperated with no trailing comma.
